## About
基于 Python3 采集天气数据，包括现在天气与天气预报，并将数据保存到csv文件

## Feature

* 采集天气数据, 接口以 https://opendata.cwb.gov.tw/api/v1/rest/datastore/{dataid}?Authorization={apikey}
* 支持现在天气与天气预报数据选择
* 存储数据为csv文件

## Requires
Python 3.11.0  
pandas 1.5.2  
requests 2.28.1  

## Usage
```
python main.py
```
